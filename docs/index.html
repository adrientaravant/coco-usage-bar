Built for the glance.

The top bar stays tiny: provider icons plus percent used. Open it when you need reset times, monthly tokens, or estimated spend.

Local by default.

Coco reads Codex and Claude Code usage from local state and existing credentials. There is no hosted account system or analytics endpoint.

Easy to share.

Teammates can download the DMG, or build from source with ./script/install.sh --login. Releases include a Sparkle appcast for updates.
