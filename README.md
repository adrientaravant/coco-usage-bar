# Coco Usage Bar

A tiny macOS menu bar app that shows your Codex and Claude Code usage at a glance.

**[Download for macOS](https://github.com/adrientaravant/coco-usage-bar/releases/latest/download/CocoUsageBar.dmg)** · [Website](https://adrientaravant.github.io/coco-usage-bar/)

## What it shows

- Codex and Claude Code limits (5-hour and weekly) with reset times.
- 30-day token usage for each provider.
- A glanceable percentage in your menu bar; click for the full breakdown.

Everything is read locally from your own logs and credentials. No account, no server, no analytics.

## Install

1. [Download the DMG](https://github.com/adrientaravant/coco-usage-bar/releases/latest/download/CocoUsageBar.dmg).
2. Open it and drag **Coco Usage Bar** into your Applications folder.
3. On first launch, right-click the app and choose **Open** (the public build is free-signed, so macOS asks once).

Requires macOS 14 or later. Updates install automatically through Sparkle.
